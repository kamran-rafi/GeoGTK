
# GeoGTK

GeoGTK is Geo related software being built using C-Language. It uses the following libraries as of now.

* GTK Library for UI.
* libcurl for HTTP communication.
* cJSON for parsing JSON data.
* Weather API


## Features

- Weather Data
- (More Data Sections Coming)
- Multiple Themes
- Cross platform (Coming Soon)
